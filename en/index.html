Freely connect process spaces* across multiple computers, just like science fiction!

The "process space" refers to the processor and memory resources etc.

Connect servers, PCs, smart phones, and web browsers together as one - no need to select a single platform. Share the process space among the connected computers and freely pass the running application between machines.

The process space changes flexibly according to the environment. There may be times where you need to distribute processing of an application across a number of computers, while at other times you want to continue processing an application offline on a single computer. (Under active development)
